Ecommerce Sales Analysis Dashboard
Overview :
This project focuses on analysing an e-commerce sales dataset to understand business performance, customer behaviour, and profitability trends. The goal was to transform raw data into meaningful insights using Excel, SQL, and Power BI.
Tools & Technologies
•	Excel – Data cleaning and processing
•	SQL – Data analysis and querying
•	Power BI – Dashboard creation and visualization
•	Gamma – PPT creation
Project Workflow
1.	Data Cleaning (Excel)
•	Handled missing values and inconsistent formats
•	Standardized data formats
•	Created calculated fields (e.g. Profit Margin, Discount Buckets)
2.	Data Analysis (SQL)
•	Performed queries to analyse
o	Sales trends 
o	Customer segmentation 
o	Regional performance
o	Product/category profitability 
•	Identified key Business patterns
3.	Data Visualization (Power BI)
•	Built as interactive one-page dashboard
•	Included KPI’s, trends, and performance metrics
•	Enabled filtering by region and sub-category
4.	Presentation (Gamma)
•	Built Presentation using Gamma AI 
Dashboard Features 
•	KPIs
o	Total Sales 
o	Total Profit
o	Total Orders
o	Profit Margin
•	Visualizations
o	Monthly Sales Trend
o	Sales by Region
o	Category and Sub-category Performance
o	Top 10 Customers 
o	Discount Impact Analysis
•	Interactive Filters
o	Region
o	Category
o	Segment
Key Insights
•	High discounts have negative effect on profitability 
•	Technology category generates the highest revenue
•	A small group of customers contributes the large share of total sales
•	Certain sub-category consistently produces losses
•	West region leads in overall sales performance
